# Calendar
日历组件

## 使用方法
``` js
var calendar1 = new calendar({
                              ele : '',
                              year : ,
                              month : ,
                              url :  '',
                              });
```
> 参数说明 

1.`ele` ：需要日历的容器

2.`year` ：年份

3.`month` ：月份

4.`url` : 获取备忘录的数据的请求地址

year和month是可选参数，没传则默认为当前时间。
